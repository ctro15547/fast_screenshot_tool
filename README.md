# fast_screenshot_tool 安卓快速截图工具

说明：
通过 adb 开放屏幕获取通道 -》 使用ffmpeg连接通道并且获取截图

使用方法：

D:\fast_screenshot_tool.exe -h

  -L string
        授权码
  -ST float
        （不建议调整性能问题可能崩溃) (default 0.025)
  -ffmpegpath string
        ffmpeg路径
  -fps int
        每秒截取帧数（不建议调整性能问题可能崩溃) (default 5)
  -port string
        api服务端口 (default "15580")
  -sn string
        设备序列号
